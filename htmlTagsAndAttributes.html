HTML: Tags & Attributes

Here is an example of the anatomy of an HTML element:

Elements are the 'boxes' of your html page, they will contain other boxes, which will contain content, such as a title, an image or a pragraph of text. Tags show what sort of element you are dealing with. An attributes gives extra information about an element. Some examples of elements are: 


            <!-- Headings -->
            <h1>This is a big heading!</h1>
            <h3>This is a smaller heading!</h3>
            <h6>This is an even smaller heading!</h6>

            <!-- Unordered list -->
            <ul>
              <li>This is on your list </li>
              <li>And this is on your list </li>
              <li>And this is also on your list </li>
            </ul>

            <!-- Paragraphs -->
            <p>Here you can write a very long paragraph of multiple lines because
            that is what paragraphs are for, right? The next paragraph will start on a
            new line.<p>

            <!-- Div -->
            <div>
              <h1>This is a div </h1>
              <p>You can put multiple elements in here that belong together. For
              example you can create an info section, a header or make sure a picture
              and an discription always stick together!</p>
            <div>

Attributes are used to give out information over you element. For example, you can give them a class.


          <!-- Attributes - class -->

          <div class="header">
            Here you'll put code for your header.
          </div>

A class can be used to name elements separatly, for example for styling purposes.
