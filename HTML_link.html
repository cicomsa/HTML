HTML: Links

The value assigned to the href attributes decides where your user should go after clicking the link. In this case we make use of a relative path: the browser will look for the file that is displayed, go one directory 'up' and will look there for an index.html file.

If you want to link to a file in the same directory, by using a relative filepath, this is how you'd do that:


                  <a href="./IntroToCSS.html">This is what you see!</a>

You can of course also link to a website that is already online, by using an URL:


                  <a href="https://www.codaisseur.com/">This is what you see!</a>
