CSS - positioning

Inline vs Block elements

Block elements take in a full line on your webpage. Examples of block elements are:

  • <p>
  • <div>
  • <footer>
See more on MDN

Inline elements don"t start on a new line and only take up as much width as necessary. Examples are:

  • <a>
  • <img>
  • <span>
See more on MDN

The Position Property

You can position your "boxes" (elements) on the page by using the CSS position property:


    header {
      position: fixed;
      top: 0;
    }

    p {
      position: static;
    }

    img {
      position: relative;
      top: -20px;
      left: 20px;
    }

    .nav {
      position: absolute;
      top: 120px;
      right: 0;
    }
  • Fixed sticks your element to a place on your screen
  • Static is the default value
  • Setting the top, right, bottom and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
  • An absolutely-positioned element is positioned relative to its parent

Some examples of the different sorts of positioning can be found on learnlayout
